CSS的样式特性

  1. 继承性

    文本样式和字体样式的继承

    绝大多数文本字体样式都可以被继承

    a标签不会继承字体颜色

  2. 堆叠性

    为一个标签定义多个样式规则

    如果样式属性不冲突，都会做用到这个标签上

  3. 优先级

    默认优先级：

    高 内联样式

    中 内部样式，外部样式 — 就近原则

    低 浏览器默认样式

    默认情况下，浏览器的开发工具中是按照默认优先级的顺序从上往下显示

  4. 调整优先级

    设置最高优先级 !important，比内联的优先级都高

    写在某个值和分号之间，!important与值之间有空格

    注意：内联样式不允许设置 !important

  3. 某某[属性^=属性值]

伪类选择器

  1. 常用的伪类选择器

    :first-child 第一个子元素

    :last-child 最后一个子元素

    :nth-child() 选中第n个元素

    关于:nth-child(）中的特殊值：

    • n 第n个 n的范围 0 到 正无穷（全选）
    • even | 2n 选中偶数位的元素
    • odd | 2n+1 选中奇数位的元素

    ==以child结尾的是在所有子元素中选择==

    :first-of-type 第一个子元素

    :last-of-type 最后一个子元素

    :nth-of-type() 选中第n个元素

    ==以type结尾的是在相同元素中选择==

  2. 否定伪类

    :not() 将符合条件的元素去除

  3. 元素的伪类

    :link 表示未访问过的a标签

    :visited 表示访问过的a标签 （只能修改连接的颜色）

    ==以上两个伪类是超链接独有的==

    :hover 鼠标移入后元素的状态

    :active 鼠标点击后，元素的状态

  4. 伪元素选择器

    ::first-letter 表示第一个字母

    ::first-line 表示第一行

    ::selection 表示选中的元素

    ::before 元素开始的位置前

    ::after 元素结束的位置后

选择器权值

选择器权值代表选择器的重要程度。

重要程度高则优先显示

  • !important > 1000

  • 内联样式 1000

  • id选择器 100

  • 类选择器 10

  • 伪类选择器 10

  • 元素选择器 1

  • 通用选择器 0

  • 继承的样式 无

权值的特点：

  1. 当一个选择器含有多个选择器时，需要将所偶有选择器的权值进行相加，值最大的显示
  2. 有多个选择器权值一样大，按就近原则
  3. 添加了!important当前css样式直接获得最高优先级
  4. 群组选择器不相加，各算各的
  5. 选择器权值的计算，不会超过本选择器权值的最大数级

CSS盒模型

盒模型包括什么

  • 外边距(margin) - 清除边框外的区域，外边距是透明的
  • 边框(border) - 围绕在内边距合内容外的边框
  • 内边距(padding) - 清除内容周围的区域，内边距是透明的
  • 实际内容(content) - 盒子的内容，显示文本和图像

标准盒模型（W3C的标准盒模型）

在标准盒模型中，width指content部分的宽度

怪异盒模型（IE盒模型）

在怪异盒模型中，width指content+padding+border的宽度

box-sizing的使用

我们默认使用的正是W3C标准盒模型，而盒模型的选取更倾向于项目合开发者的习惯。如果想要切换盒模型，需要借助box-sizing属性。

光标

定位

注意：元素一旦设置了position，并且取值为relative/absolute/fixed其中一种时：

  1. 这个元素被称为已定义元素

  2. 解锁四个方向的定位属性：top/right/bottom/left

    左右冲突会以left为准

  3. 会解锁堆叠属性z-index

  • 相对定位

    position:relative配合偏移属性

    不脱离文档流，不影响周围元素

    如果一个元素设置了相对定位，并且不写偏移属性，这个元素对页面没有任何影响

    如何偏移：相对自己原来的位置偏移

    使用场景：

    1. 做元素位置的微调
    2. 作为绝对定位的祖先级已定位元素

  • 绝对定位

    position:absolute配合偏移属性

    绝对定位，脱离文档流

    如何偏移：

    1. 如果元素没有已定位的祖先元素，绝对定位元素相对body左上角进行偏移
    2. 以最近的已定位的祖先级元素左上角做偏移，子绝夫相

    使用场合：页面布局中使用，可以让页面移动到页面的任何位置

  • 固定定位

    position:fixed配合偏移属性

    如何偏移：相对于body左上角

    使用场合：把元素永远固定在页面可视区域的一个位置

  • 堆叠顺序

    • HTML代码中，后写的元素堆叠顺序高
    • z-index：默认值 大于0小于1
    • 浮动在第几层 小于0 大于-1
    • 只有已定位的元素才能设置堆叠
    • 堆叠顺序对父子元素无效，儿子永远压着父亲

Sass 和 Less

区别和选择

  • Sass（Scss）：
    • 使用缩进格式，强调代码的缩进和层级
    • 代码块使用缩进表示
    • 文件扩展名为 .sass或.scss
  • Less：
    • 使用css样式的语法，类似于原生Css
    • 代码块使用大括号表示
    • 文件扩展名为.less

变量和混合（Mixin）

  • Sass（Scss）:
    • 使用$符号定义变量
    • 支持@mixin定义混合
  • Less：
    • 使用@符号定义变量
    • 支持.mixin()定义混合

其他差异

  • Sass（Scss）：
    • 使用!default关键字定义变量的默认值
    • 内置了很多函数，如颜色计算、字符串处理等
  • Less：
    • 使用@arguments传递所有传递给Mixin的参数
    • 功能相对较少，需要使用插件来扩展
