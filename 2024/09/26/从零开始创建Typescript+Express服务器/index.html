前言

为了搭建一个nodejs + express + typescript + 热加载的一个简易项目, 需要配置一些基础的文件来确保 ts 代码能够被正常编译和正确运行。

1. 安装 Node.js

确保系统上已经安装了一个 Node.js 项目即可

2. 初始化项目

创建一个项目目录并进入, 然后使用 npm 初始化一个新项目。

3. 安装依赖

安装Express 和Typescript作为项目依赖, 并安装 @types/express 和 @types/node 作为开发以来, 这些类型定义会帮助我们在Typescript 代码中获取自动补全和类型检查:

4. 配置 TypeScript

使用tsc --init命令快速生成一个默认的配置文件, 并根据需要进行修改设置。

这会生成一个包含默认设置的tsconfig.json文件, 我们可以根据需要修改这些设置。

在这个配置中, 我们指定输出目录为dist, 源代码目录为src, 并启用了严格类型检查。

7. 编译并运行TypeScript代码

在命令行中通过npm run build命令来编译TypeScript代码。这将会根据tsconfig.json配置文件中的设置, 将src目录下的.ts文件编译成JavaScript文件, 并且输出到dist目录, 然后再通过npm start命令来启动Express应用。
