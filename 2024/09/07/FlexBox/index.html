FlexBox

前言

为了能够实现以任何可以想象的方式创建漂亮且响应式的布局, 我们必须定义一个 参考点 。它可以是 用于布局的正文或特定容器。 每个包含任何类型内容的HTML元素都可以是FlexBox或者Grid的容器。

因此我们的第一步就是找出如何在其父元素内的任意位置对齐元素( 顶部、底部、左侧、右侧、以及中心和它们之间的每个组合 )。

如果我们需要使用FlexBox布局时, 只需要在父元素中加入display:flex属性即可。

这是, 页面将会从纵向排列改为这样的横向排列 :

这是因为 大多数元素默认有块或者内联的样式, 每个默认块元素占据整个屏幕的宽度, 因此其它元素会被挤到下一行中。当我们使用FlexBox布局时, 就会忽视这种默认行为。

接下来, 为了观察容器与子元素的关系, 我们为父元素添加一个边框:

在FlexBox布局中, 每个应用这个属性的元素都会由两条轴控制, 分别为主轴(Main Axis)和`交叉轴(CrossAxis)。

如图, 绿色的先即为主轴, 红色的线为交叉轴。

如果我们想要将元素定位在主轴上的任何位置, 我们可以通过CSS属性justify-content来实现:

justify-content: flex-start

justify-content: flex-end

justify-content: center

主轴的问题解决了, 接下来就需要去解决交叉轴居中的问题。为了能够让我们看出来交叉轴上发生了什么, 首先我们将container的高度设置一下:

与主轴同理, 如果我们需要控制元素在交叉轴上的位置, 我们则需要使用CSS属性align-items。

align-items: flex-start

align-items: flex-end

align-items: center

至此, 如何将元素居中的问题已经解决了, 那就是使用FlexBox布局 :

因为FlexBox灵活的特点, 即使我们改变视口的大小, 盒子的位置始终将处于容器的中心处。

其它更多属性

justify-content

CSS属性justify-content还有其他几个重要的属性:

  • space-between ==> 将空间间隔均匀分布, 并且第一个和最后一个元素将接触布局边缘

  • space-around ==> 与 space-between 相似, 但第一个和最后一个元素将不会接触布局边缘, 并且元素间的间隔不同

  • space-evenly ==> 与 space-around 相似, 但元素与元素之间的间隔是相同的

与justify-content相同, 那些属性同样可以使用在交叉轴上。但是现在我们只有一行元素, 这将会导致所有元素都位于交叉轴起点。

flex-direction

这个属性是用来控制主轴的方向的。并且由于justify-content和align-items是依据主轴和交叉轴进行改变位置的, 因此这个属性也会影响到其使用效果。

flex-direction: row 默认值 主轴从左向右 交叉轴从上至下

flex-direction: row-reverse 主轴从右向左 交叉轴从上至下

flex-direction: column 主轴从上至下 交叉轴从左向右

**flex-direction: column-reverse ** 主轴从下至上 交叉轴从左至右

gap

gap属性的作用很简单, 就是给予处于容器中的子元素间隔。

它拥有两个参数: 行间距 和 列间距

如果只传入一个参数, 则默认控制二者。我们可以通过 row-gap 和 column-gap 来进行分别控制。

flex-wrap

这个属性用来控制子元素是否换行。

它拥有两个参数:

  • wrap: 换行 ==> 当子元素宽度溢出容器时,将溢出部分切换到下一行,是默认值

    将子元素box的宽度修改为300px后, 父元素容不下5个box, 因此第五个box换行至下一行

  • no-wrap: 不换行 ==> 即使子元素溢出容易也不换行
